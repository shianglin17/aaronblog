## 為什麼進行這次變更？

為了更好地管理和維護不斷增長的 API 文檔，我們採用了模組化的文檔結構。這種方式有以下優點：

1. 每個 API 有獨立的文檔，便於專注和查找
2. 減少單一文檔的體積，提高可讀性
3. 便於與 Postman Collection 整合
4. 支持團隊協作開發
