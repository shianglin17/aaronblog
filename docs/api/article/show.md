# 獲取單篇文章 API

取得單篇文章的詳細資料。

## 請求資訊

**請求方法**
```
GET /api/article/{id}
```

**請求標頭**
```
Accept: application/json
Authorization: Bearer {token}
```

## 請求參數

| 參數 | 類型    | 必填 | 說明              |
|------|---------|------|-------------------|
| id   | integer | 是   | 文章 ID，必須大於 0 |

## 錯誤回應

### 1. 資源不存在 (404)
```json
{
    "status": "error",
    "code": 404,
    "message": "文章不存在",
    "meta": {}
}
```

### 2. 無效請求 (400)
```json
{
    "status": "error",
    "code": 400,
    "message": "無效的請求",
    "meta": {}
}
```

### 3. 未授權 (401)
```json
{
    "status": "error",
    "code": 401,
    "message": "未授權或授權已過期",
    "meta": {}
}
```

### 4. 伺服器錯誤 (500)
```json
{
    "status": "error",
    "code": 500,
    "message": "伺服器內部錯誤",
    "meta": {}
}
```
